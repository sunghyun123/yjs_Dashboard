API 명세·투입실적 데이터 구조 (엑셀 명세 반영)

  • 전송 방식: 기존 ERP 등록 방식대로 POST + JSON으로 보냅니다.
  • 레코드 구조: work_code(지중 No), input_date(투입일) + 품목별 주간/야간 필드를 한 레코드로 전송합니다.
