# yjs_Dashboard UX QA Checklist

## 1) 공통 사전 조건

- 서버 실행: `uvicorn main:app --reload`
- 브라우저에서 세션 로그인 상태 확보
- 관리자 테스트는 `role=admin` 계정 사용

## 2) 입력 화면 (`/`)

- [ ] 채팅 카테고리(`schedule_create/memo/other/update_request/delete_request`) 선택이 가능하다.
- [ ] `schedule_create`로 문장을 전송하면 AI 응답 후 등록 확인 카드가 표시된다.
- [ ] 등록 확인 버튼 클릭 시 성공 메시지가 표시된다.
- [ ] `memo` 카테고리 전송 시 메모 등록 접수 메시지가 표시된다.
- [ ] `other/update_request/delete_request` 카테고리 전송 시 관리자 큐 접수 메시지가 표시된다.
- [ ] 사진 버튼 클릭 후 파일 선택 시 카테고리 모달이 열린다.
- [ ] 카테고리 선택 후 전송 시 업로드 성공 메시지와 폴더/파일명이 표시된다.
- [ ] 응답 후 상태 칩(요청 접수/등록 확인 대기/조회 결과)이 카드 형태로 표시된다.

## 3) 상황판 (`/dashboard.html`)

- [ ] 로그인 세션이 없으면 입력 화면으로 리다이렉트된다.
- [ ] 검색어, 날짜, 오늘만 토글 필터가 동작한다.
- [ ] 일정 카드 클릭 시 액션 패널이 열리고 `작업 인원 추가/수정/삭제` 버튼이 표시된다.
- [ ] 일정 카드 `수정` 클릭 시 즉시 수정 모달이 열리고 기존 값이 채워진다.
- [ ] `즉시 수정` 실행 후 성공 메시지가 표시된다.
- [ ] 일정 카드 `삭제` 클릭 시 확인 창이 표시되고 즉시 삭제 성공 메시지가 표시된다.
- [ ] `상세`가 2줄 이하면 카드에서 바로 보이고, 3줄 이상이면 `상세 보기` 버튼이 표시된다.
- [ ] `상세 보기` 클릭 시 모달에서 전체 상세가 표시된다.
- [ ] 외출/행선표 인원은 관리자 등록 목록 기준으로만 표시된다.
- [ ] `외출` 상태에서만 행선/복귀시각 UI가 보이고, `사무실/야간작업`에서는 숨겨진다.
- [ ] 복귀 시각 모달에서 시간 선택 및 `▲/▼` 30분 단위 조절이 동작한다.
- [ ] 60초 주기 자동 갱신 중에도 외출/메모 섹션이 함께 갱신된다.

## 4) 관리자 화면 (`/admin.html`)

- [ ] 로그인 사용자가 admin이 아니면 접근이 차단된다.
- [ ] 필터(상태/요청자/기간) 조회가 동작한다.
- [ ] 새로고침 후 필터 값이 유지된다(localStorage).
- [ ] 요청 카드에 요청 유형 칩, 상태 배지가 올바르게 표시된다.
- [ ] 후보찾기 결과에서 `선택` 버튼으로 일정 ID가 입력된다.
- [ ] 승인 버튼으로 요청이 처리되고 목록이 갱신된다.
- [ ] 반려 버튼 클릭 시 반려 모달이 열리고 사유 입력 후 반려 처리된다.
- [ ] 백업데이터 생성 버튼 클릭 시 실행 모달이 열리고 대상 날짜 입력 후 실행된다.
- [ ] `외출/행선표 인원 관리` 카드에서 인원 등록/삭제가 정상 동작한다.

## 5) 레거시 경로 (`/board.html`)

- [ ] `/board.html` 접근 시 `/dashboard.html`로 리다이렉트된다(307).

## 6) 회귀 확인

- [ ] `/`, `/dashboard.html`, `/admin.html` 정적 페이지 응답이 정상이다.
- [ ] `/board.html`은 리다이렉트 응답이 정상이다.
- [ ] `/api/schedules/today`, `/api/schedules/execute`, `/api/admin/requests/review` 호출이 정상이다.
- [ ] `python -m pytest -q` 실행 시 기존 스모크 테스트가 통과한다.
